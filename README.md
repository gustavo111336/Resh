## 🌌 O QUE É O RESH?
O **RESH** é um interpretador de comandos customizado, focado em performance, estética e simplicidade. Ele nasceu do zero no Termux para dar ao usuário o controle total sobre o sistema, com aquele toque **Chronos** de elite!!!! kakejskksjskskdjkd ;-;

### ✨ FUNCIONALIDADES DE ELITE:
- 📂 **Prompt Dinâmico:** Mostra o caminho real com estética Neon.
- ⚙️ **Suporte ao .reshrc:** Carrega seus alias e configs automaticamente! JURO!!!!
- 📜 **Histórico Persistente:** Tudo o que você digita fica salvo no `.resh_history`.
- 🎯 **Fast Execution:** Busca direta no $PATH do sistema.
- 🧬 **Perfeição Binária:** Versões otimizadas para caberem em exatos **512 bytes** (Tamanho de um MBR!) JURO VEY!!!!
