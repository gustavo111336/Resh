## ⚡ O QUE É O CROSH-TERMUX?
O **crosh** é uma interface customizada para Termux que simula o ambiente do **ChromeOS Flex (Ultra Edition)**. Focado em desenvolvedores de Kernel e entusiastas de sistemas bare metal, ele transforma seu terminal em uma estação de trabalho estilizada e eficiente.

> [!IMPORTANT]
> Este projeto utiliza o **SHC** para compilar scripts em binários ELF, garantindo maior portabilidade e execução sem dependências de shell externas.

---

## 🛠️ FUNCIONALIDADES NEON
- 🔵 **Interface Chronos:** Prompt estilizado e cores ANSI vibrantes.
- 🏗️ **Kernel Workflow:** Atalhos integrados para compilação ARM32 (`kbuild`).
- 🕹️ **QEMU Ready:** Emulação rápida de Cortex-A15 com um único comando (`krun`).
- 🔋 **System Tools:** Simulação de comandos nativos do ChromeOS como `battery_test`.
- 📦 **Zero Libs:** Graças ao `shc`, o binário roda limpo no seu `$PREFIX/bin`.
